# Genesis Forge

Genesis Forge turns a rough idea, uploaded brief, or source URL into a readable project plan with starter files.

## What Works

- Create a project from an upload, text idea, or URL.
- Watch simple generation progress without internal workflow jargon.
- View the generated summary, app structure, screens, and starter code.
- Download the work immediately as PDF, Markdown, JSON, or a starter ZIP.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The app is built with TanStack Start, React, Vite, Nitro, and Tailwind CSS. It stores generated projects in browser localStorage, so there is no backend required for the current version.
